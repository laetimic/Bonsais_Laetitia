Conversion_Heic_jpg

Explication

Ce code permet de convertir automatiquement les fichiers .heic en format .jpg.

Le code balaie le dossier et remplace les fichiers .heic en .jpg et supprime les images .HEIC.

Ce code peut etre exécuté en préparation d’une relance de la génération des pages html

# Lister tous les fichiers HEIC dans le dossier
fichiers_heic <- list.files(path = chemin_dossier, pattern = "\\.HEIC$", full.names = TRUE, ignore.case = TRUE)


# Vérifier si des fichiers HEIC ont été trouvés
if (length(fichiers_heic) == 0) {
  print("⚠️ Aucun fichier HEIC trouvé dans le dossier.")
  } else {
  print("📂 Fichiers HEIC trouvés :")
  print(fichiers_heic) } # Affiche la liste des fichiers trouvés

# Boucle pour convertir chaque fichier
for (fichier in fichiers_heic) {
  
  # Lire l'image HEIC
  img <- image_read(fichier)
  
  # Nouveau nom de fichier en .jpg (remplace l'extension)
  nouveau_fichier <- sub("\\.heic$", ".jpg", fichier, ignore.case = TRUE)
  
  # Sauvegarder en format JPG
  image_write(img, path = nouveau_fichier, format = "jpg")
  print(paste("✅ Converti :", fichier, "→", nouveau_fichier))
  
  # Supprimer le fichier HEIC après conversion
  file.remove(fichier)
  print(paste("🗑️ Supprimé :", fichier))
}
